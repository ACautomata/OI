# 二分答案
解题的时候往往会考虑枚举答案然后检验枚举的值是否正确。若满足单调性，则满足使用二分法的条件。把这里的枚举换成二分，就变成了“二分答案”。
>白了说就是暴力优化
大致的模板
```C++
int find()
{
	int l=1,r=1000000001;
	while(l+1<r)
	{
		int mid=(l+r)/2;
		if(check(mid))
			l=mid;
		else
			r=mid;
	}
	return l;
}
```
